正和、零和、負和遊戲解釋

Posted by 虎斑貓 on Wednesday, July 7, 2021

正和、零和、負和遊戲解釋

簡單名詞解釋

正和遊戲：

所有贏家所獲取的總和，比起輸家失去的總和還多。

例如，一場遊戲裡面，所有贏家的總收益是2000萬，輸家的總收益是-500萬。

甚至沒有輸家，大家都是贏家。

零和遊戲：

所有贏家所獲取的總和，等同於輸家失去的總和。

你賺的每一塊錢，一定是在同樣遊戲中，某人失去的一塊錢。

負和遊戲：

所有贏家所獲取的總和，小於輸家失去的總和。

那缺少的部分會跑哪裡去了？

有人做莊，莊家拿走差額。

遊戲的進行會有成本摩擦，在過程中損耗掉。

不過負和遊戲不是這次要討論的主題。

那麽股市是正和還是零和遊戲？

老話一句，看你用什麼樣的『時間』去看待。

道理很簡單，想想看，為什麼老闆每個月才發一次薪水？

因為你工作了一個月，才有比較明顯可以衡量的產出和勞動結果。

公司將所有資產的產出結果，放到市場上賺取利潤，再將一部分的收入支付必須成本，其中，包括你的薪水。

公司為什麼願意支付薪水給員工，因為扣除成本，公司還有賺錢。

公司持續會賺錢，擁有的總資產在很理想的狀況下，是不斷地上升，而股票代表著的是公司資產的擁有權利。

所以不考慮其他情況之下，股價(公司總資產)，隨著時間上升是理所當然。

但是，這是需要『時間』的啊！

不可能老闆交辦你一件事情，可以像變魔法一樣，瞬間完成給他，於是公司的收益也不可能憑空生出來。

投資的概念也是一樣，如果短暫的時間就可以得出結果，根本不是什麼投資，因為投資也需要『時間』。

或者，你也可以說，投資的概念本身就建立在時間之上。

因此投資股市，單就基本原理，是一件非常正向的事情，在這個正和遊戲裡面，大家參與資本市場勞動帶來的結果，人人都有機會分配到收益，而這些收益是由大家勤奮的工作而來，正正當當且扎扎實實。

所以基本上會講投資，常見都是半年、一年、N年的產出結果，勞動需要花費和付出時間。

反之當天買賣，或者短時間買賣，你能講出在這麼短的時間內，實體經濟上這間公司在這麼短的時候，能夠給這個世界帶來什麼嗎？三天前和三天後的狀況大體上半斤八兩，所以它就變成一個零和遊戲。

( 試著想想看三天前的你，和三天後的你，對於用你一輩子人生的角度來看，到底是有差別到哪裡去。 )

在總體狀況不變的情況下，有人會賺錢，那麼那些賺的錢，就是有人買單。

造就必定有輸家和贏家的情況。

那為什麼大家還是如此熱衷參與這樣的遊戲呢？是因為資本市場本身就是個『不完全效率』的存在，客觀的存在資訊為何？每個個體認知都有落差，而大家也很容易認為自己拿到的資訊是有優勢的，看到了獲益的機會，因此想搞投機一把。

結果就是總有人開心，總有人流淚。

坦然的態度

對於投資或者投機，沒人說只能兩者選一。

大部分情況下，這兩者也容易混在一起，混在一起發生。

舉例來講，我看好某公司的新產品在未來半年會賣得不錯，買進了持股，這半年公司將產品賣得好，當然有大筆實質利潤進來。

不過遇到市場喜歡這個話題，人們就一窩蜂的炒作，把1年後、2年後、3年後的利潤都事先算進去了，股票的價格因此一路往上飆，偏離公司總資產實際價值的價格越來越多。

那麼半年後，我賣出了手上的持股，你認為這是投資還是投機呢？

又或者是在玩正和遊戲還是零和遊戲？有時候很難說得清楚。

投機不是錯，好的投機操作，勝過數年投資的收獲。

但千萬不要投機失敗，騙自己在投資，了解自己在做什麼，才能嘗試評估風險，有風險的心理的預期，晚上才睡得好。

平常心看待進入市場的操作，也只有這樣才能夠長久參與。

「真诚赞赏，手留余香」
